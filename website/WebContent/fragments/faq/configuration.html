Yes you can.
Sloeber creates a arduinoPlatform folder where all downloaded info is stored.
If you upgrade using a new install next to your current install you can copy or move the arduinoPlugin folder.
If you only want to reuse the downloaded files only copy the arduinoPlugin/downloads folder over. You can also put the arduinoPlatform folder on a different location on you platform by creationg the environment variable SLOEBER_HOME and point it to the parent of the arduinoPlugin folder.
Yes you can and should.
In the settings page a private library folder is requested. This setting should point to a location where you downloaded Arduino libraries.
If you haven't downloaded Arduino libraries yet do as follows:
  1. Consider using the Arduino IDE. (strongly advised)
  2. Point it to a location where you can download Arduino libraries.
